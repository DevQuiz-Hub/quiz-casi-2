# quiz-casi-2 🧠

Interactive self-assessment test engine for **CASI (Calidad y Auditoría de los Sistemas de Información / Information Systems Quality and Auditing)**.

This repository contains an extended self-assessment platform and curated question set created by the repository owner with AI-assisted content generation, validation and refinement workflows.

Part of the DevQuiz-Hub ecosystem.

## 📚 Core Topics Covered

* **Block 1**: Information Security Management Systems (ISMS), CIA Triad, security policies, information assets, information classification and security governance.
* **Block 2**: IT Governance, Corporate Governance, ISO/IEC 38500, CISA certification, compliance principles, risk analysis and internal control frameworks.
* **Block 3**: IT Auditing, digital evidence, chain of custody, forensic analysis, cybercrime, audit methodologies and security assessment processes.
* **Block 4**: Business Continuity Planning (BCP), Disaster Recovery Planning (DRP), backup strategies, asset recovery and organizational resilience.
* **Block 5**: Network Security, firewalls, IDS, VPN technologies, OSI model security controls and access control mechanisms.
* **Block 6**: Vulnerability Assessment, fingerprinting, Nmap, reverse engineering, DAST methodologies and offensive security concepts.
* **Block 7**: Web Application Security, XSS, CSRF, authentication mechanisms, session management and secure system design.
* **Block 8**: SDLC, information assurance, logging systems, risk management and information protection strategies.

## 🛠️ Data Contract & Logic

This repository implements the standardized architecture of the organization:

* **`casi_questions.json`**: Holds the structured question schema, metadata and explanatory content.
* **`quiz.js`**: Core asynchronous engine handling in-place shuffling, dynamic data fetching and client-side state management.
* **`index.html`**: Zero-dependency SPA frontend built with responsive modern CSS variables.
